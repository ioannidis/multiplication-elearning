Teacher Help

Student management

Display student academic performance

By clicking on the "Details" button you can see the progress of each student, ie in which courses he has attended and in which he has been successfully examined, as well as the grade he received.

Edit student details

Pressing the "Edit" button can edit the student's basic information. Pressing "Save" saves the changes, one pressing "Cancel" cancels and closes the dialog box.

Delete student

Clicking on the "Delete" button to delete the student from the application.

Lesson management

Display lesson details

By clicking on the "Details" button, you can see the basic details of each lesson.

Edit lesson details

By clicking on the "Edit" button, you can edit the basic details of each lesson. Pressing "Save" saves the changes, on pressing "Cancel" cancels and closes the dialog box.
